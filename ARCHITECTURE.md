# TikTok 爆款内容引擎 — 架构文档

## 项目概览

这是一个 TikTok 爆款内容分析和脚本生成工具。用户输入一个关键词（比如"眼镜"），系统会自动抓取 TikTok 上该关键词下的热门视频，用 AI 分析这些爆款的共同规律，然后生成可以直接使用的短视频脚本，最后还能生成数字人口播视频。

## 核心概念词汇表

| 术语 | 类比解释 |
|------|----------|
| Apify | 一个"爬虫机器人"服务，帮你自动浏览 TikTok 并把视频数据收集回来，就像派一个助手去书店帮你抄下畅销书排行榜 |
| Groq Whisper | 一个"语音转文字"服务，能把视频里说的话变成文字，就像一个超快的速记员 |
| DeepSeek | 一个 AI 大模型，负责"思考"——分析爆款规律、生成脚本，就像请了一个短视频运营专家帮你总结和创作 |
| A2E (Avatars to Everyone) | 一个"数字人"服务，给它一段文字，它能生成一个虚拟人对着镜头念出来的视频 |
| Hook | 视频开头的"钩子"，就是前 3 秒抓住观众注意力的话，比如"你知道吗？90%的人眼镜都戴错了" |
| CTA (Call to Action) | 视频结尾引导观众行动的话，比如"关注我，下期告诉你答案" |
| Caption | TikTok 视频下方的文案（通常是标签），不是口播内容 |
| Transcript | 口播转录文本，就是视频里真正说出来的话 |

## 如果我是用户，发生了什么

```
1. 我打开网页，看到一个搜索框
2. 我输入"眼镜"，点击"开始分析"
3. 页面显示进度条：
   ⬛⬛⬜⬜⬜ 正在抓取 TikTok 热门视频...
   ⬛⬛⬛⬜⬜ 正在转录视频口播内容...（新增步骤）
   ⬛⬛⬛⬛⬜ AI 正在分析爆款规律...
   ⬛⬛⬛⬛⬛ 正在生成脚本...
4. 我看到爆款视频卡片（封面、播放量、文案）
5. 我看到 AI 的分析报告（爆款 Hook 类型、内容结构、情绪策略）
6. 我看到 3 条生成的脚本（每条有 Hook、正文、CTA）
7. 我可以选一条脚本，生成数字人视频预览
```

## 技术选型

| 技术 | 为什么选它 | 备选方案 | 不选备选的原因 |
|------|-----------|----------|---------------|
| 单文件 HTML + React CDN | 零配置，打开就能用 | Next.js / Vite | Demo 阶段不需要构建工具 |
| Apify TikTok Scraper | 最成熟的 TikTok 数据获取方案 | 自己写爬虫 | TikTok 反爬很强，自己写成本太高 |
| Groq Whisper | 免费额度大，速度极快（比 OpenAI 快 10x） | OpenAI Whisper API | OpenAI 要钱，Groq 免费且更快 |
| DeepSeek V3 | 性价比极高，中文能力强 | GPT-4 / Claude | 贵 10-100 倍，DeepSeek 够用 |
| A2E | 有 100 免费 credit | HeyGen / D-ID | 其他都需要先付费 |

## 架构图（数据流动）

```
用户输入关键词（如"眼镜"）
        │
        ▼
┌─────────────────────┐
│ 模块一：数据获取      │  Apify TikTok Scraper
│ (shouldDownloadVideos │  → 返回视频列表 + 下载 URL
│  = true)             │
└─────────┬───────────┘
          │ videos[] (含视频文件 URL)
          ▼
┌─────────────────────┐
│ 模块 1.5：语音转录    │  Groq Whisper API
│ (Top 5 视频)         │  → 视频文件 → 文字
└─────────┬───────────┘
          │ videos[] + transcript 字段
          ▼
┌─────────────────────┐
│ 模块二：爆款分析      │  DeepSeek V3
│ (文案+口播+数据指标)  │  → 分析报告 JSON
└─────────┬───────────┘
          │ analysis JSON
          ▼
┌─────────────────────┐
│ 模块三：脚本生成      │  DeepSeek V3
│ (基于分析结果)        │  → 3 条脚本
└─────────┬───────────┘
          │ scripts[] JSON
          ▼
┌─────────────────────┐
│ 模块四：视频生成      │  A2E 数字人 / 图文预览
│ (选一条脚本)         │  → 视频 URL
└─────────────────────┘
```

## 可插拔配置

```javascript
const config = {
  data: "apify",         // 数据源
  transcription: "groq", // 转录: groq | none
  llm: "deepseek",       // LLM: deepseek

};
```
